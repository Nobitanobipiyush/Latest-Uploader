## Quick Start (Local/VPS)

1) Create `.env` from `.env.example`
2) Install: `pip install -r requirements.txt`
3) Run: `python main.py`

## Render Deploy
Set Environment Variables: `BOT_TOKEN`, `API_ID`, `API_HASH`
Start command: `python main.py`
